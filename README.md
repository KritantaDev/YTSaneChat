# YTSaneChat

Quality filter for youtube streams.

Written for Technoblade's livestreams.

### Things it does

* Filters out non-member chat
* Filters out messages >30% capital letters
* Filters out messages that are only emojis

### Installation

Pick a directory, any of them work

`git clone https://github.com/KritantaDev/YTSaneChat.git`

Open chrome, click the icon on the far top right, open extensions. 

Enable Developer mode, click load unpacked, navigate to your copy of this repository, and click open

Enjoy :)
